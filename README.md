# KGMRPC


---

### 🌟 Features

* **Private Mode:** Disable information about game name
* **Disable Presence:** Disable KoGaMa Discord Presence.
* **Cross-Region:** One app for any region game.
* **Auto Inject:** App detects kogama automatically
